## Overview

This project is based on the Nutritionist Light Theme Healthy Diet Planning Template by Praha.
It shows how to build a simple and clean responsive website using only HTML and CSS, following what we learned in class.

## Features ## 

Fully responsive on desktop, tablet, and mobile

Built with Flexbox and Grid layouts

Includes skip link, alt text, and good color contrast for accessibility

Smooth hover animations on buttons and icons

Clean and valid HTML and CSS code

Simple layout: Hero, Features, Blog, Testimonials, Pricing, and Footer sections
